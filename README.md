# Blitz Bat Works

Custom taped wiffle ball bats.

## Editing the site

Content is edited via [Pages CMS](https://pagescms.org) — no code needed.

1. Go to https://app.pagescms.org
2. Sign in with GitHub
3. Open the `blitzbatworks` project
4. Edit fields → click **Save**
5. Site auto-updates in ~30 seconds at https://blitzbatworks.com
